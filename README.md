
# 🚗 Car Sales Analysis Dashboard

## 📊 Project Overview

The **Car Sales Analysis Dashboard** is a data analytics project developed using **Power BI** to analyze car sales performance and identify key business insights.
This dashboard helps understand sales trends, brand performance, and regional sales distribution.

The project demonstrates **data cleaning, visualization, and dashboard design skills** using real-world sales data.

---

## 🎯 Objectives

* Analyze total car sales performance
* Identify top-performing car brands
* Understand regional sales distribution
* Track monthly and yearly sales trends
* Provide insights for business decision-making

---

## 🛠 Tools & Technologies

* Power BI
* Microsoft Excel / CSV Dataset
* Data Visualization
* Data Analysis

---

## 📂 Dataset

The dataset contains information about:

* Car Brand
* Model
* Sales Amount
* Region
* Date of Sale
* Dealer Information

This data was used to create interactive visualizations in Power BI.

---

## 📈 Dashboard Features

The Power BI dashboard includes:

* Total Sales KPI
* Sales by Car Brand
* Sales by Region
* Monthly Sales Trends
* Top Selling Car Models
* Interactive Filters and Slicers

These visualizations allow users to explore the data dynamically.

---

## 🔍 Key Insights

* Some car brands contribute significantly to total sales.
* Certain regions show higher car sales demand.
* Monthly trends highlight peak sales periods.
* Sales performance varies across different car models.
